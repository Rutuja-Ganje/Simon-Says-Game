Simon-Says-Game:
A fun memory game where players repeat an increasing sequence of colors. Built using HTML, CSS, JavaScript to practice DOM manipulation and game logic.


## How to Play on PC / Laptop
1. Open index.html in your browser.  
2. Press any key to start the game.  
3. Watch the sequence of colors and repeat it by clicking the buttons.  
4. The sequence gets longer each level. Game over if you make a mistake!
